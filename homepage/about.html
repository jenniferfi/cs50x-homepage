Criteria for this project

This page must contain at least four different .html pages, at least one of which is index.html, and it should be possible to get from any page on your website to any other page by following one or more hyperlinks. The site should look nice on browsers both on mobile devices as well as laptops and desktops.

In addition, the site must use the following technologies.

HTML

Use at least ten (10) distinct HTML tags besides <html>, <head>, <body>, and <title>.

Bootstrap

Integrate one or more features from Bootstrap into your site.

CSS

Have at least one stylesheet file of your own creation, which uses at least five different CSS selectors, and within which you use a total of at least five different CSS properties.

JavaScript

Integrate one or more features of JavaScript into your site to make your site more interactive.
